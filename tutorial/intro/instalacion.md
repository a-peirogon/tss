# Instalación

## Requisitos

- Python 3.11 o superior
- pip

## Pasos

1. Clona el repositorio o copia `apuntes.py` en tu proyecto.
2. Instala las dependencias:

```bash
pip install markdown pyyaml pygments
```

3. Crea la estructura de carpetas y edita `conf.yaml`.
4. Ejecuta:

```bash
python apuntes.py build
```
